# BJJ Mat IQ

A comprehensive BJJ match analysis and progress tracking tool built with Streamlit.

## Features

- 🥊 **Match Review**: Detailed match analysis with event logging and tactical assessment
- 👤 **Athlete Management**: Individual athlete profiles and progress tracking  
- 📈 **Trends Analysis**: Performance trends across multiple matches
- 📊 **History**: Chronological match and assessment history
- 📄 **Export**: PDF and Word document generation for reports

## Live Demo

🚀 **[Try BJJ Mat IQ Live](https://bjj-mat-iq.streamlit.app)**

## Installation

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Usage

1. **Select an Athlete** from the dropdown (or create a new profile)
2. **Start a New Match Review** to analyze a BJJ match
3. **Log Events** during match playback with timestamps
4. **Complete Assessment** of technical areas
5. **View Trends** and export reports

## Technology Stack

- **Frontend**: Streamlit
- **Data Processing**: Pandas
- **Visualizations**: Plotly
- **Export**: ReportLab (PDF), python-docx (Word)
